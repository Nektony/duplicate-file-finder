# Changelog

All notable changes to Duplicate File Finder will be documented in this file.

## [9.0.1] - April 2026

### Fixed
- Minor refinements and bug fixes

## [9.0] - April 2026

### Improved
- Redesigned interface
- Select by Folders
- Full VoiceOver support
- Stability

## [8.5.3] - February 2026

### Improved
- Duplicate overview
- Performance and stability improvements

## [8.5.2] - November 2025

### Improved
- Duplicate search in local iCloud folders
- Search for duplicate photos and videos in Photos libraries
- Handling of duplicate files without extensions

## [8.5.1] - September 2025

### Added
- Support for VoiceOver screen reader

### Improved
- Interface enhancements for compatibility with macOS 26 beta

## [8.5] - August 2025

### Added
- Ability to export duplicate scan reports to CSV

### Improved
- Speed of similar photo and audio search
- Saving and restoring of duplicate scan sessions
- Performance of auto-selection for large duplicate sets
- Similar photo and audio search now follows file extensions from Skip List settings
- Refined the application settings window
