# Duplicate File Finder

> Find and remove duplicate files to free up disk space on your Mac.

[![macOS](https://img.shields.io/badge/macOS-11.0%2B-blue)](#system-requirements)
[![Version](https://img.shields.io/badge/version-8.5.3-green)](#changelog)
[![App Store](https://img.shields.io/badge/Mac_App_Store-★★★★★-brightgreen)](#)

Duplicate File Finder scans your Mac, external drives, and cloud storage to find exact duplicates and similar files — then helps you safely remove them and reclaim disk space.

## Key Features

**Exact Duplicates** — Find files with identical content (bit-for-bit match), regardless of file name or location.

**Similar Photos & Audio** — Detect visually similar photos and near-identical audio tracks, even if metadata or bitrate differs.

**Duplicate & Similar Folders** — Identify folders with identical contents, or similar folders that share some files but also have unique ones.

**Merge Folders** — Combine the contents of two or more folders: merge any folders, merge similar folders found during scan, or merge multiple folders at once.

**Compare Folders** — Compare two specific folders side by side to see which files are duplicates and which are unique.

**Compare Files** — Check whether two specific files are exact duplicates or different.

**Select Duplicates in Folders** — Quickly select all duplicates within a specific folder, or mark a folder to keep (star icon) so duplicates in other folders are selected instead.

**Smart Selection** — Auto-select duplicates for removal based on configurable rules. Skip list lets you exclude folders, files, and extensions.

**Safe Removal** — Files go to Trash by default. Last file in a group is always protected. Restore removed duplicates with one click via removal history.

**Photos & Music Library Support** — Scan Photos.app libraries and Music.app libraries directly for duplicates.

**External & Cloud Storage** — Scan external drives (USB, NAS, SMB) and local iCloud folders.
